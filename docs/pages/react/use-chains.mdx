# useChains [Hook for getting configured chains]

## Import
```tsx
import { useChains } from '@zerodev/waas'
```

## Usage
```tsx
import { useChains } from '@zerodev/waas'

function App() {
  const chains = useChains()
}
```

## Parameters

No parameter needed.

## Return Types

```tsx
import { type UseChainsReturnType } from '@zerodev/waas'
```
