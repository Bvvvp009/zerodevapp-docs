# useChainId [Hook for getting current connected chain id]

## Import
```tsx
import { useChainId } from '@zerodev/waas'
```

## Usage
```tsx
import { useChainId } from '@zerodev/waas'

function App() {
  const chainId = useChainId()
}
```

## Parameters

No parameter needed.

## Return Types

```tsx
import { type UseChainIdReturnType } from '@zerodev/waas'
```

`number`
