# Sudo Policy

The sudo policy gives full permission to the signer.  The signer will be able to send any UserOp, or sign any messages, or both.

## API

```ts
import { toSudoPolicy } from "@zerodev/permissions/policies"
import { PolicyFlags } from "@zerodev/permissions/constants"

const sudoPolicy = toSudoPolicy({
  policyFlag: PolicyFlags.VALIDATE_ALL,
})

const validator = toPermissionValidator(publicClient, {
  entryPoint,
  signer: someSigner,
  policies: [
    sudoPolicy,
    // ...other policies
  ],
})
```

- `policyFlag` can be:
  - `VALIDATE_ALL`: this is the default value.  Gives the signer full permission.
  - `VALIDATE_ONLY_SIG`:  the signer can sign messages, but not userops.
  - `VALIDATE_ONLY_USEROP`:  the signer can sign userops, but not messages.
