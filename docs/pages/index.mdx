# ZeroDev Smart Wallet

ZeroDev is a modular toolkit for building smart wallet experiences, particularly optimized for DeFi.  Some of the things we offer:

- Fully non-custodial signin methods including EOAs, socials, and passkeys.
- Easy onramp & offramp.
- Gas abstraction: pay gas in any ERC20 tokens, and even sponsor gas for users.
- One-click trading and transaction automation.
