# ZeroDev Smart Wallet

ZeroDev is a smart embedded wallet for Web3 projects, especially DeFi projects.  By leveraging account abstraction (AA), ZeroDev enables you to build safe, friendly, and powerful experiences for your users.

ZeroDev focuses on the following areas:

- Onboarding: through on-chain passkeys and social signins
  - signing
  - importing assets: easily help EOA users deposit assets into your DApp-specific smart accounts

- Transaction
  - Gas abstraction
  - Batch execution
  - One-click trading
  - Transaction automation
  - DeFi integrations
  - Parallel orders

- Security
  - recovery
  - multisig

- Offramp & interoperability
  - wallet connect
  - export assets
