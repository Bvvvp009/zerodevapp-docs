# Signin

With ZeroDev, your users can create smart accounts in three ways:

- Passkeys
- Socials
- EOAs

## SDK

### Passkeys

```ts
const passkeyValidator = createPasskeyValidator({
  // ...
})
```

### Socials

```ts
const socialValidator = createSocialValidator({
  // ...
})
```

### EOAs

```ts
const socialValidator = createEcdsaValidator({
  // ...
})
```

## React Hooks

```ts
const { validator } = useWaaSValidator({
  options: ['passkeys', 'socials'],
})
```
